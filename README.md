# Greenline Tennis Club

Scaffold for a premium Wimbledon-inspired tennis club marketing website built with Next.js 14 App Router, TypeScript, Tailwind CSS, and shadcn/ui foundations.

## Tech Stack

- Next.js 14.2.21
- React 18.3.1
- TypeScript 5.6.3
- Tailwind CSS 3.4.17
- shadcn/ui config
- Framer Motion animations
