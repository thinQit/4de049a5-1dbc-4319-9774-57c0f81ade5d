# Anand Tennis Club

Next.js 14 + TypeScript + Tailwind + shadcn/ui scaffold for **Anand Tennis Club** (Wimbledon-inspired green and gold brand).

## Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui
- Zod
- Framer Motion
- NextAuth-ready env setup
- PostgreSQL-ready env setup
